## Advanced Usage

To seperate handlers from the plugin definition, you can use the `definePlugin` utility function:

```ts filename="definePlugin.ts"
import { definePlugin } from '@hyperse/wizard';

const deployPlugin = definePlugin({
  name: 'deploy plugin',
  localeMessages: helpMessages,
  setup: (wizard, pluginCtx) => {
    return wizard;
  },
});
```
