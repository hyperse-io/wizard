# About Hyperse

Hyperse is an open-source full-stack development suite that provides front-end components, architecture patterns, and tech solutions. Built with `TypeScript`, `GraphQL`, and `Node.js`.

## Related Projects

Hyperse is powered by these incredible open source projects:

- https://github.com/hyperse-io/wizard
- https://github.com/hyperse-io/wizard/tree/main/packages/wizard-plugin-loader
- https://github.com/hyperse-io/wizard/tree/main/packages/wizard-plugin-version
- https://github.com/hyperse-io/wizard/tree/main/packages/wizard-plugin-error
