# @hyperse/wizard-plugin-completion

A plugin based on Fig for automatic command completion, which is compatible with various scenarios, including use in Fig and Amazon Q.

## Installation

```bash
# npm
npm install --save-dev @hyperse/wizard-plugin-completion

# yarn
yarn add --dev @hyperse/wizard-plugin-completion

# pnpm
pnpm add --save-dev @hyperse/wizard-plugin-completion
```
