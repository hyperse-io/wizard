Public API - WizardWithUse

Important: The createWizard function returns a WizardWithUse instance, not a full Wizard instance. This means external users can only access the six core properties: use, on, parse, name, description, and version.

Core Properties

The WizardWithUse type exposes only the essential public interface:

use(plugin) - Plugin Registration

Registers a plugin with the wizard instance. This is the primary way to extend functionality.

import { createWizard } from '@hyperse/wizard';

const wizard = createWizard({ name: 'my-cli' });
wizard.use(myPlugin); // âœ… Available

on(event, listener) - Event Listening

Registers event listeners for various wizard lifecycle events.

import { createWizard } from '@hyperse/wizard';

const wizard = createWizard({ name: 'my-cli' });
wizard.on('build', () => console.log('Building...')); // âœ… Available

parse(options?) - Command Execution

Parses command line arguments and executes the appropriate command.

import { createWizard } from '@hyperse/wizard';

const wizard = createWizard({ name: 'my-cli' });
wizard.parse(); // âœ… Available

name - Wizard Name

Gets the name of the wizard instance.

import { createWizard } from '@hyperse/wizard';

const wizard = createWizard({ name: 'my-cli' });
console.log(wizard.name); // âœ… Available - outputs: 'my-cli'

description - Wizard Description

Gets the description of the wizard instance.

import { createWizard } from '@hyperse/wizard';

const wizard = createWizard({
  name: 'my-cli',

Type Safety

The WizardWithUse type ensures type safety by:

  1. Restricting Access: Only the six core properties are accessible from outside
  2. Plugin Integration: The use method properly types plugin integrations
  3. Event Handling: The on method provides type-safe event handling
  4. Method Chaining: All methods return WizardWithUse for fluent chaining
  5. Read-only Properties: name, description, and version are read-only for external access
import { createWizard } from '@hyperse/wizard';

const wizard = createWizard({
  name: 'my-cli',
  description: 'A powerful CLI tool',
  version: '1.0.0',
});

// âœ… Allowed - Core methods and properties
wizard.use(myPlugin).on('build', handler).parse();
console.log(wizard.name, wizard.description, wizard.version);

// âŒ Not available - Internal methods
// wizard.register(command);     // Type error
// wizard.flag('debug', {});     // Type error
// wizard.interceptor(handler);  // Type error
